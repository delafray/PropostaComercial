# Instruções para Alterar os Direitos Autorais (Ofuscados)

Este sistema utiliza ofuscação simples (Base64) para proteger os textos de direitos autorais.

## Localização do Arquivo
O arquivo responsável é: `src/utils/core_lic.ts`

## Como Alterar Manualmente

1.  Acesse um site de "Base64 Encoder" (ex: base64encode.org).
2.  Digite seu novo texto.
3.  Copie o resultado codificado.
4.  Substitua o valor entre aspas no arquivo `src/utils/core_lic.ts`.
